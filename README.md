## react_+antd后台管理系统搭建

#### 1.创建项目

```
create-react-app mymanager 
```

####2. 安装所需工具

```
yarn add axios less-loder react-router-dom antd
```

####3. 打开webpack配置,暴露webpack配置

```
yarn eject 
```

```
出现报错Build fails after eject: Cannot find module '@babel/plugin-transform-react-jsx' 
```

解决方法

```
1.删除 node_modules 文件夹
2.重新 npm start
3.运行
```
